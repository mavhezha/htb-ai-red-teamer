# Applications of AI in InfoSec

HTB Academy module covering practical machine learning applications in cybersecurity.
Four models were built, trained, and evaluated against live API endpoints.

## Labs

| Lab | Algorithm | Accuracy |
|-----|-----------|----------|
| Spam Detection | Naive Bayes + CountVectorizer | 91% |
| Network Anomaly Detection | Random Forest | 100% |
| Malware Image Classification | ResNet50 + PyTorch | 70.8% |
| Skills Assessment | Naive Bayes + CountVectorizer | 100% |
